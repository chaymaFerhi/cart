# Test Plan – OpenCart Register Feature
**Project Name:** OpenCart Manual Testing
**Module Name:** Register
**Test Type:** Manual Testing

## Test Environment

* Browser: Google Chrome
* OS: Windows
## Objective

To verify that the cart functionality works correctly when adding, updating, and removing products.
## Test Approach

Testing the "Register Account" functionality.


## Scope

* Add product to cart
* Add multiple products
* Update quantity
* Remove product
* Empty cart
* Verify cart total calculation
## Out of Scope

* Payment functionality
* Checkout functionality

## Test Deliverables

* Test Cases
* Bug Report
* Screenshots
* Test Summary
